# 🧩 Algorithm Study

> Consistency is the key to mastery.

LeetCode problem solving repository written in Java.  
Aims to build strong problem-solving skills through daily practice.

---

## 🛠 Language
- Java

---

## 📌 Goals
- Solve at least one problem every day
- Understand and internalize core algorithms and data structures
- Write clean, readable, and efficient code

---

## 📂 Structure
Problems are automatically organized by LeetHub.
Each folder contains the solution file and a description of the problem.
```
LeetCode/
└── problem-title/
├── problem-title.java
└── README.md
```

- Hash Map

- Binary Search
